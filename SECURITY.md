# Security Policy

## Supported Versions

This is a **public brand showcase repository** only.  
It contains no application code, no credentials, no infrastructure and no customer data.

| Version | Supported |
|---------|-----------|
| main    | ✅ Brand assets only |

## Reporting a Vulnerability

If you discover any security-related issue related to the agentemIA brand, website or services (not this repo):

1. **Do not** open a public issue
2. Email immediately: **security@agentemia.it** (or info@agentemia.it)
3. We will acknowledge within 24 hours and provide a timeline

## Data & Privacy

- All production data of agentemIA resides exclusively in the European Union
- Full GDPR compliance
- agentemIA acts as Data Processor; the client remains Data Controller

Thank you for helping keep agentemIA secure and trustworthy.
